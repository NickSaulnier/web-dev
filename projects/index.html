User Interface Design with React.js

This project—a collection of user interfaces created with React.js—showcases my work for CS7580 (User Interface Design) at Northeastern University. The interfaces include: a job search application, note taking application, discussion board app, and trivia game. The emphasis is on event-driven, object-oriented user interfaces that are interactive, responsive, and accessible.

Automatic Text Summarization of Privacy Policies using Latent Semantic Analysis

This project is a prototype of an automatic text summarization system for privacy policies. I used tf-idf scoring of OPP-115 privacy policy corpus annotations and latent semantic analysis to extract topical sentences from pre-segmented privacy policies, combining the results into a overall summarization for a given input privacy policy. The project included a web scraper to scrape pre-segmented privacy policies from The Usable Privacy Policy Project

Clustering CALIPSO Level 2 5km Cloud Layer Lidar Data by Surface Elevation and Cloud Layer Base Altitude using a Self-Organizing Map

This project explores atmospheric cloud typing using surface elevation and cloud layer base altitude. A data extractor and dataset generator was first created to format a directory of CALIPSO cloud lidar data as a CSV. The project notebook contains exploratory analysis of the input data features, code for performing feature selection using principal feature analysis, code to perform clustering and cluster visualization with a self organizing map, and analysis of the generated clusters.

Neural Metaphor Identification with BiLSTMs and GloVe Embeddings

This project presents an exploration of current state of the art techniques for detecting metaphorical language using word embeddings and deep learning. I used the VU Amsterdam Metaphor Corpus as training and testing data, and compared several BiLSTM models that leveraged GloVe and locally trained embeddings, as well as dot product attention mechanisms.
